Projetos antigos

  • Greenpyce, uma biblioteca em python para fazer pré processamento e engenharia de variáveis utilizando Pandas
  • Wally IPS, um painel web simples que processava dados de posicionamento interno (IPS), assumindo a aplicação em um Shopping, e gerava recomendações de lojas para os clientes e gráficos sobre a frequência visita às lojas
  • Tio Pytinhas, um modelo para calcular a quantia em dinheiro em uma foto de um conjunto de moedas. Foi feito para rodar em um Raspberry Pi + webcam. Construído em cima do dataset de moedas brasileiras.
  • Introdução à programação, aulas online utilizando a linguagem Processing e ensinando os conceitos enquanto um jogo simples é construído.
