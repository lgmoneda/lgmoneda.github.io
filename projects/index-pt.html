Proust Dataset

Construindo e explorando um conjunto de dados para o romance "Em busca do tempo perdido" de Marcel Proust. Dois conjuntos de dados foram extraídos de ebooks. Há versões em português, inglês, csv por parágrafos e texto todo em arquivo txt.

  • Código Fonte: Github
  • Blog Post: Tweetando como Proust

Tio Pytinhas

Identificando moedas e calculando o valor em reais de um conjunto de moedas através de imagens. Dois conjuntos de dados foram construídos utilizando uma webcam: moedas únicas e conjunto de moedas. Ambos foram feitos em um ambiente controlado, no qual o afastamento e fundo eram fixos.

  • Código Fonte: Github

Wally IPS

Um painel web simples para ser alimentado por dados de uma aplicação com IPS (Indoor Positioning System). O projeto foi realizado pensando em uma aplicação que fornecesse dados com horário, data e localização de um usuário. Como caso especifico, decidiu-se imaginar um Shopping Center. Os dados foram gerados artificialmente.

  • Código Fonte: Github.
  • Linguagens: Python, HTML, JQuery e CSS.
  • Frameworks: Flask.
  • Outras tecnologias: Pandas e MySQL.
